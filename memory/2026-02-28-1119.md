## 🇵🇹 Português de Portugal - Pontos Fundamentais

### PALAVRAS QUE MUDAM: PT-BR vs PT-PT

| PT-BR | PT-PT |
|-------|-------|
| você | tu |
| obrigado/a | obrigado/a |
| você sabe | sabes |
| você entende | entendees |
| celular | telemóvel |
| ônibus | autocarro |
| fone | telemóvel |
| chute | chute |
| puxar | puxar |
| empurrar | empurrar |

### CONJUGAÇÃO VERBAL

Em Portugal usamos frequentemente **ter** em vez de **haver** no presente do indicativo:

- PT-BR: "Há uma pessoa à porta"
- PT-PT: "Temos uma pessoa à porta"  

MUITO usado em conjugações como:

- "Não *tenho* nada para dizer" (não *quero* dizer nada)
- "Tenho problemas" (tenho / tenho de / tenho que)

### VOCABULÁRIO PT-PT

- **Chinelos** - chinelos
- **Frangueiro** - frangueiro (carrinho de compras)
- **Botijão de Gás** - botijão de gás
- **Jantar** - jantar
- **Almoço de domingo** - almoço de domingo (às 14h)
- **Sondar a água** - verificar se a água está limpa
- **Casa de banho** - casa de banho
- **Autocarro** - autocarro

### FRASES TÍPICAS

- "Bom dia!" (não **"Boa tarde"** como saudação de dia)
- "Por favor" (não **"Combinado"** - isso é PT-BR)
- "Já vou" / "Já estou a ir" (não **"Já vou embora"** neste sentido)
- "Bons dias"
