# 2026-01-30 & 2026-01-31

## Eventos e Decisões Técnicas
- **Migração para OpenClaw:** O sistema foi atualizado para a versão 2026.1.29 (OpenClaw). A estrutura de pastas mudou de `~/.clawdbot` para `~/.openclaw`.
- **Otimização Extrema da VM:** Removida a interface gráfica (LXQt, X11) e pacotes desnecessários (LibreOffice, Bluetooth, etc.). O servidor agora corre em modo "Headless", libertando ~1.5GB de disco e baixando o uso de RAM em repouso.
- **Ativação da Groq:** Configurada a API da Groq com o modelo Llama-3.3-70B como motor principal para conversas rápidas e estável, resolvendo erros de quota do Gemini.
- **Ligação ao Oura Ring:** Integrado o acesso via API do Oura Ring. O token pessoal foi guardado de forma segura em `.oura_token`.
- **Criação de Skill:** Desenvolvida a skill `saude-total` para gerir o protocolo de longevidade.
